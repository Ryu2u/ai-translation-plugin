# AI 翻译插件 - 设计规格书

**日期**: 2026-04-22
**状态**: 已批准

---

## 概述

开发一款 Chrome/Edge 浏览器扩展插件，使用 Vite + Vue 3 + TypeScript 构建，集成 AI 大模型 API 实现网页翻译功能。

## 目标用户

- 需要在浏览网页时快速翻译内容的用户
- 有自备 API Key 的开发者/高级用户

---

## 技术栈

| 技术 | 用途 |
|------|------|
| Vite | 构建工具 |
| Vue 3 | UI 框架 |
| TypeScript | 类型安全 |
| Chrome Extensions API | 浏览器扩展能力 |

---

## 功能设计

### 1. 翻译触发入口

| 入口 | 描述 |
|------|------|
| 右键菜单 | 页面任意位置右键，选择翻译选项 |
| 悬浮按钮 | 右下角固定按钮，点击翻译整页 |
| 选中文字气泡 | 选中文字后出现翻译小按钮 |

### 2. 翻译结果展示

| 方式 | 描述 |
|------|------|
| 原文下方插入 | 翻译结果直接插入到原文下方，高亮显示 |
| 侧边栏展示 | 右侧滑出翻译面板，可拖拽调整 |

### 3. 翻译引擎

- **API 格式**: OpenAI 兼容格式
- **默认功能**: 仅用于翻译
- **多 Key 支持**: 可配置多个 API Key 并切换使用

### 4. 设置页面 (Options Page)

- API Key 管理（添加/删除/切换）
- 默认目标语言设置（默认中文）
- UI 风格选择（深色主题）
- 快捷键配置

---

## UI 风格

**深色主题** - 暗黑配色，适合夜间使用，保护眼睛

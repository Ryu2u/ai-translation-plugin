AI 翻译插件 - 整体架构

基于 Vite + Vue + Chrome Extensions 构建

技术栈

Vite Vue 3 Chrome Extensions TypeScript

核心功能入口

🖱️
右键菜单

页面任意位置右键
选择翻译功能

📌
悬浮按钮

右下角固定按钮
点击翻译整页

💬
选中文字气泡

选中文字后
出现翻译按钮

系统架构

┌─────────────────────┐ │ Content Scripts │ │ (悬浮按钮/气泡) │ └─────────┬───────────┘
│ 监听用户交互
▼
┌─────────────────────┐ │ Background Worker │ │ (消息转发/处理) │ └─────────┬───────────┘
│ API 调用
▼
┌─────────────────────┐ │ OpenAI API │ │ (翻译引擎) │ └─────────────────────┘

设置页面 (Options Page)

  • API Key 管理（添加/删除/切换）
  • 默认目标语言设置
  • UI 风格选择（深色主题）
  • 快捷键配置

优势

  • Vite 快速构建，HMR 开发体验好
  • Vue 3 响应式 UI，开发效率高
  • 深色主题，护眼舒适
  • 多 API Key 切换，灵活方便

注意

  • 需要用户自备 API Key
  • 仅支持 Chromium 内核浏览器
